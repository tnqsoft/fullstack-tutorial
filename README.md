# fullstack-tutorial
Fullstack tutorial for PHP developer

Install API
- Required Symfony, Composer
- MySql
- Restore DB from demo-api/db.sql
```
cd demo-api
composer install
symfony sever --allow-http
```

Install APP
- Required NodeJS, Angular CLI, Typescript
```
cd demo-app
npm i -d
ng serve
```

Goto browser
http://localhost:4200
